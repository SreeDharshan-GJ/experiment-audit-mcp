# Scientific Rules

Examples

Rule 001

IF

Multiple hyperparameters changed

AND

Claim == Ablation

THEN

Confounded Experiment

Confidence High

--------------------------------

Rule 002

IF

Only one random seed

THEN

Low Statistical Confidence

--------------------------------

Rule 003

IF

Validation loss increases

AND

Training loss decreases

THEN

Possible Overfitting

--------------------------------

Rule 004

IF

Gradient norm explodes

AND

Loss diverges

THEN

Possible Gradient Instability

--------------------------------

Rule 005

IF

Learning rate changed

AND

Baseline changed simultaneously

THEN

Comparison Invalid
