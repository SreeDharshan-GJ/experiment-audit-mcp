# Scientific Reasoning Engine

## Mission

Experiment Audit is a scientific reasoning engine for machine learning experiments.

It does NOT summarize experiments.

It evaluates scientific evidence and produces defensible conclusions.

The LLM is only the interface.

Experiment Audit performs the reasoning.

## Principles

Every conclusion must

- be supported by evidence
- include confidence
- explain why
- cite supporting observations
- identify contradictions
- state limitations

No unsupported claims.
